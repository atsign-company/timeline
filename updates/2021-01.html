Here are the fun apps that will be demoed:

  4. @Buzz: A peer-to-peer social networking application that enables individuals to present different personas to different people and receive personalized offers from companies they trust

Now for the Techies!

For developers - Virtual Environment 2.01 is launched and available for developers at VE2.01. The main difference is the focus on flutter developers and being able to get up and running in as few steps as possible. We have instructions for Linux, MacOS, and of course WIndows.

Non repudiation of public data is now available via metadata, and it will also be available for encrypted data in the coming month.

We continue to add and update the developer libraries we have published to pub.dev (currently we have 14 packages published) and we see the popularity %’s going up so we know folks are building with them, which is very exciting to see! Most notably, we have added at_contact and at_contacts_flutter as well as at_chat_flutter libraries to make it easy for applications to implement shared contact data and chat messages.

@operations - Last month's move NS1.com has proved to be a great move and while we have had behind the scenes outages they have been routed around by the NS1 technology so no-one would ever know. We have also had Google rebuild, without warning, our Kubernetes clusters (root server, directory & registrar), no doubt for their own operational issues. All that engineering effort was worth it, everything was fully autonomously handled. We have had some customer affecting problems, specifically with our automated scripts for activating @signs - we continue to improve our monitoring and error handling scripts to ensure that all the services we run have no outages from a customer viewpoint.

@core protocol - The sync 2.0 code is now in production, this is a complete rewrite of the original sync code and incorporates all the learnings we had from using the 1.0 codebase. The sync improvements are on the secondaries so there is no need to recompile app code.

Our next focus is the notification subsystem, which will bring many to many notifications and further improvements. These changes have been driven by developers of the @rrive app, which is great. Each new application teaches us more about what needs additional build out.

We wish you the very best 2021!
